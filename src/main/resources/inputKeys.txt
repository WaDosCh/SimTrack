Input keys
F1 = show this guide ingame
F2 = show coordinates debug view
F3 = select pathfinding tool
F4 = spawn a train
F5 = toggle hex grid
F6 = toggle profiler
F12 = end game
ESC = deselect current tool
1-9 = select tools in hotbar

Buildtool
Q/E = turn current object
Tab = Mirror current object
Click = Place tile
RightClick = Remove tile
